# shelter-now-frontend

현재 위치 기준으로 가까운 대피소와 쉼터를 지도형 UI와 리스트로 함께 탐색하는 프론트엔드 MVP다.

## 현재 구현 범위
- 유형별 쉼터 맵
- Leaflet 기반 실제 지도 타일/마커 탐색
- 무더위쉼터 / 한파쉼터 / 민방위대피소 / 임시주거시설 탐색
- 검색 / 유형 필터 / 자치구 필터
- 운영 상태 필터
- 정렬(거리순/수용인원순/이름순)
- 현재 위치 기준 재조회
- 즐겨찾기 쉼터 저장
- 거리순 리스트
- 쉼터 상세 카드
- 외부 지도 열기 / 전화하기

## 로컬 실행
```bash
npm install
npm run dev
```
